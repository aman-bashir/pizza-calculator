# Pizza Dough Calculator

A baker's-percentage pizza dough calculator. Pick a style (Neapolitan, New York, Sicilian, Detroit, and more), set the number of doughballs, hydration, salt, oil, sugar, room temperature and fermentation time — and get exact ingredient weights. Yeast is auto-adjusted for temperature and proof time.

**Live site:** https://aman-bashir.github.io/pizza-calculator/

## Files
- `index.html` — the site root served by GitHub Pages.
- `pizza-calculator.html` — identical copy.

Single self-contained HTML file, no build step. Open locally in any browser or use the live link above.
